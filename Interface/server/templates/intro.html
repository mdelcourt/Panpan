Western PanPan

Western PanPan is a web application allowing the analyis of western experiments images. Each western (a line of blots which have been migrated together) is analysed to determine the intensity of peaks after removal of the background for this specific western. Results are given for each independent western as a csv file containing values of intensities and as images representing the different steps of analysis.

How to use Western PanPan?

Western PanPan's input is a png image containing one or more western experiments. A western experiment is understood as a line of blots (refered to as "western" hereafter). (TODO pic) The file needs to be .png (not .jpg nor .tiff, etc...) and should ideally not have been previously saved with a lossy compression (eg. jpg) then converted to png.

Automatic detection of westerns is not yet implemented in Western PanPan. Therefore, the user first has to manually delimit the western by a green rectangle (green value RGB: 128,255,128). This can easily be done using paint or an equivalent simple image processing program.

The user is first greeted by the interface that allows him to upload the image of the server by clicking "Browse...", selecting the image file and submiting it. TODO

For each western analysed, the results presented contain:

a false-color picture of the background,
a picture in false colors of the western,
a picture of the western with the contour of the blots (which are the regions of the western not considered as background by the algorithm),
a line plot of the horizontal profile of luminosity of the background (which is the sum of luminosities for each column of pixels of the background),
a histogram of luminosity values for all pixels and a line plot of values of blots with the limits of peaks.
